Tipos de arquivos para fotos

O que é o JPG?

JPG é um método comum usado para comprimir imagens fotográficas. O grau de redução pode ser ajustado, o que permite a você escolher o tamanho de armazenamento e seu nível de compromisso com a qualidade da imagem. Geralmente se obtém uma compressão pouco perceptível na perda de qualidade da imagem. Veja mais.

O que é o GIF?

Graphics Interchange Format ou GIF é um formato de imagem de bitmap que foi desenvolvido por uma equipe do provedor de serviços on-line CompuServe, liderado pelo cientista de computação americano Steve Wilhite em 15 de junho de 1987. Veja mais.

O que é o PNG?

A sigla PNG significa “Portable Network Graphics”, que é um formato de imagem digital utilizado para implementar elementos visuais em um site na web. Além disso, este formato foi desenvolvido para superar as limitações de outros formatos, como o GIF, em aspectos como cores e licenciamento. Veja mais.

O que é o SVG?

SVG é a abreviatura de Scalable Vector Graphics que pode ser traduzido do inglês como gráficos vetoriais escalonáveis. Trata-se de uma linguagem XML para descrever de forma vetorial desenhos e gráficos bidimensionais, quer de forma estática, quer dinâmica ou animada. Veja mais.
